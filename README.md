# 🏡 Airbnb Price & Review Analysis  
A data analytics project using **Python** for data cleaning and **Power BI** for dashboard visualization.  
The goal is to understand how pricing, review scores, room types, and host characteristics influence Airbnb listings.

---

## 📊 Project Overview
This project analyzes **212,000+ Airbnb listings** to uncover insights about:

- What factors influence listing prices  
- Whether review scores affect pricing  
- How Superhosts differ from regular hosts  
- Which room types generate higher revenue  
- How neighbourhood impacts pricing  
- How guest capacity affects price  

---

## 🛠 Tools & Technologies
- **Python** (Pandas, NumPy)
- **Power BI**
- **DAX**
- **Jupyter Notebook**

---

## 🧹 Data Cleaning (Python)
Key steps included:

- Removing missing/null values  
- Cleaning price column  
- Fixing incorrect data types  
- Removing duplicates  
- Standardizing categorical values  
- Creating new calculated columns  

The cleaned dataset was exported for Power BI.

---

## 📈 Dashboard Insights (Power BI)
Key KPIs:

- **Average Price:** 145.89  
- **Average Review Score:** 65.08  
- **Total Listings:** 212K  
- **Avg Price per Guest:** 59.45
  

Major findings:

- Location is the strongest price driver  
- Hotel rooms & entire places have higher prices  
- Superhosts charge slightly more  
- Review scores have moderate impact  
- Guest capacity increases revenue potential
